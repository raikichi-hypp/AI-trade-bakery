1. AIと投資データ分析とは？

AI（人工知能）というと難しく聞こえますが、実は私たちの日常生活にも使われています。

  • 🍞 パンの売上を予測するAI
  • 📈 天気データを分析して翌日の気温を予測するAI
  • 💰 株価や投資データを分析するAI

つまり、投資データをAIで分析すれば、どの銘柄が上がるか・下がるかを予測する ことが可能になるのです。

2. パン屋が投資で陥りやすい問題

投資を始めたばかりのパン屋さんがよく陥る問題は次の3つです。

  • 📊 データをどう活用すればいいかわからない
  • ⚡ 感覚で売買してしまう（例：「なんとなく上がりそうだから買う」）
  • ⏳ 投資の勉強に時間を割けない（パン作りが忙しい）

しかし、Pythonを使えば、データ分析を自動化し、効率的に投資判断をすることができます。

3. Pythonを使った簡単な投資データ分析

それでは、Pythonを使って投資データを簡単に分析する方法を紹介します。

✅ 1. Pythonと必要なライブラリをインストール

まずは、Pythonをインストールし、データ分析に必要なライブラリを準備します。

pip install pandas matplotlib yfinance

これで、データを扱う pandas、グラフを描く matplotlib、投資データを取得する yfinance が使えるようになります。

✅ 2. 株価データを取得する

次に、Pythonを使って株価データを取得してみましょう。 例えば、Apple（AAPL）の株価データを取得するには、次のコードを実行します。


import yfinance as yf

# Appleの株価データを取得
data = yf.download("AAPL", start="2023-01-01", end="2024-01-01")

# データの最初の5行を表示
print(data.head())

✅ 3. 株価の推移をグラフ化する

株価の変動を視覚的に見ることで、投資の判断がしやすくなります。


import matplotlib.pyplot as plt

# 株価の推移をグラフ化
plt.figure(figsize=(10,5))
plt.plot(data["Close"], label="AAPL Stock Price", color="blue")
plt.title("Appleの株価推移")
plt.xlabel("日付")
plt.ylabel("株価 ($)")
plt.legend()
plt.show()

このコードを実行すると、Appleの株価推移がグラフとして表示されます。

✅ 4. 移動平均線を使った投資判断

投資の判断をする際に、移動平均線を利用するのも一つの方法です。


# 移動平均線を計算
data["SMA_50"] = data["Close"].rolling(window=50).mean()

# グラフを表示
plt.figure(figsize=(10,5))
plt.plot(data["Close"], label="AAPL Stock Price", color="blue")
plt.plot(data["SMA_50"], label="50日移動平均", color="red")
plt.title("Appleの株価と移動平均線")
plt.xlabel("日付")
plt.ylabel("株価 ($)")
plt.legend()
plt.show()

赤い線（移動平均線）と株価の関係を見て、売買の判断がしやすくなります。
